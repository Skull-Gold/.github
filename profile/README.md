# 🏴‍☠️ Skull & Gold

**Skull & Gold** is a **2D RPG Point & Click pirate-themed game**, powered by **Polkadot**.  
It features:
- 🎮 Accessible mini-games  
- 💰 Collectible rewards  
- 🌍 A seasonal world that evolves over time
